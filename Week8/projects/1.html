MITutor

MITutor is an intelligent tutoring system designed to train counselling students in adhering to therapeutic protocols while interacting with clients. Motivational Interviewing is a therapeutic protocol that emphasizes change talk directed towards client behaviors while engaging in active listening skills. MITutor enables deliberate practice of therapeutic skills and automated feedback. In doing so, counseling students practice with virtual pedagogical agents that simulate dialogue. For each response provided to the agents, the systems relies on both knowledge and model tracing techniques to personalize instruction. In knowledge tracing, the system uses learning curves for each skill component to assess counseling student skill acquisition to adjust the sequence of problem solving opportunities. In model tracing, natural language processing algorithms classify student responses in regards to the type of therapeutic skill as a means to deliver feedback.
